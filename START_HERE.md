## 🚀 Getting Started in 3 Steps

### Step 1: Open Terminal & Start Frontend
```bash
cd placement-tracker-frontend
npm install
npm run dev
```
→ Frontend runs on **http://localhost:3000**

### Step 2: Open Another Terminal & Start Backend
```bash
cd placement-tracker-backend
mvn clean install
mvn spring-boot:run
```
→ Backend runs on **http://localhost:8080**

### Step 3: Set Up Database
```bash
mysql -u root -p placement_tracker < schema.sql
```
→ Database ready with sample data

**That's it! You're ready to use the app!** 🎉

## 🆘 Troubleshooting

### If frontend doesn't load
→ Check if running on http://localhost:3000
→ Check browser console (F12)
→ Verify Clerk key in .env.local

### If backend doesn't start
→ Check MySQL is running
→ Verify database credentials
→ Check port 8080 is free

### If database doesn't load
→ Check MySQL installation
→ Run schema.sql file
→ Verify database name is "placement_tracker"

See [QUICK_REFERENCE.md](QUICK_REFERENCE.md) for detailed troubleshooting.
